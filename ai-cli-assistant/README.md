# 🧠 AI CLI Assistant

A lightweight and efficient **command-line productivity tool** built with Python to manage daily tasks and notes, with simple AI-like suggestions to boost productivity.

---

## 🚀 Features

- ✅ Add, view, complete, and delete tasks  
- 📝 Create and manage personal notes  
- 🤖 AI-like suggestions based on pending tasks  
- 💾 Persistent local storage using JSON  
- ⚡ Fast and simple CLI interface

💡 How It Works
Tasks and notes are stored in a local data.json file
Users interact through a simple menu-driven CLI
The assistant analyzes pending tasks and gives basic productivity suggestions
