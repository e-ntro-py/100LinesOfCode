# 🤖 AI Terminal Assistant

A lightweight, offline AI-style command line assistant that understands simple natural language commands and performs useful tasks directly from your terminal.

---

## 🚀 Features

- 🕒 Show current time
- 📅 Show current date
- 🌐 Open websites instantly
- 📄 Create files
- 🗑️ Delete files
- 🔍 Search anything on Google
- 🧮 Built-in calculator
- ⚡ Works offline (no API required)

---

## 🧠 Example Commands

```bash
open youtube
create file notes.txt
delete file notes.txt
search python tips
calc 5+10*2
time
date
help
exit
