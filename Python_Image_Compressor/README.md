# Image Compressor

## Description
A Python utility that compresses JPG and PNG images using the Pillow library. It provides a human-readable report on the storage space saved.

## Installation
1. Activate your environment.
2. Install requirements:
   `pip install -r requirements.txt`

## Usage
1. Run: `python compressor.py`
2. Enter the path to a single image or a folder.

## Technologies
- Python 3
- Pillow (PIL)
