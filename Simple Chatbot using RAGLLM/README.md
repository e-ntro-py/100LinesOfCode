# Simple Chatbot using RAGLLM

## Description
It's a simple chatbot based on the PDF using the RAG architecture and the LLM. LLM is loaded in the file no API key is used.

## Installation
Run the code line wise to install the dependencies

## Usage
After running all the cell, at last cell gives link to the gradio which is a UI to text the chatbot
