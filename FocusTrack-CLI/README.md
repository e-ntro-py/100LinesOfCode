# 🚀 FocusTrack CLI

> A smart, minimal, and powerful command-line task manager built with Python to boost productivity and organize your workflow efficiently.

---

## 🌟 Overview

**FocusTrack CLI** is a lightweight yet powerful task management system designed for developers, students, and productivity enthusiasts.
It allows users to manage tasks efficiently with features like priority-based sorting, search functionality, completion tracking, and persistent storage.

Built with simplicity and scalability in mind, this project is perfect as a foundational productivity tool and can be extended into a full-stack or AI-powered system.

---

## ✨ Features

* ✅ Add tasks with priority levels (Low / Medium / High)
* 📋 View tasks with clear status indicators
* 🎯 Mark tasks as completed
* 🗑 Delete tasks easily
* 🔍 Search tasks by keyword
* 🔃 Sort tasks based on priority
* 📊 View statistics (Total, Completed, Pending)
* 💾 Persistent storage using JSON
* ⚡ Fast and lightweight CLI interface
